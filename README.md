#Aplikasi To Do List
##Deskripsi
Aplikasi Sederhana berbasis PHP menggunakan Laravel untuk mencatat tugas Harian.

##Fitur
-Tambah Tugas.
-Tandai tugas selesai.
-Hapus Tugas

##Struktur Folder
-Asset/public/css/style.css
      /public/js/script.js
#Base Layout
-views/'index.blade.php'

##Cara Menjalankan
-Clone git ke 'htdocs/'
-Jalankan XAMPP dan buka di 'http://127.0.0.1:8000'
